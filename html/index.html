RC_Driver 문서화

프로젝트 개요

RC_Driver는 STM32G474RET6 마이크로컨트롤러 기반의 14V / 10A PMSM(영구자석 동기 전동기) 인버터 드라이버입니다.
RC카용 브러시리스 모터(BLDC/PMSM)를 FOC(Field Oriented Control, 자속 지향 제어) 방식으로 구동하며, Hall 센서 기반 위치/속도 추정과 센서리스 확장 역기전력(EEMF) 옵저버를 지원합니다.

제어 알고리즘

FOC(Field Oriented Control) 방식으로 d-q 동기 좌표계에서 전류를 독립적으로 제어합니다.
Hall 센서 기반 PI-PLL로 전기각과 속도를 추정하며, 센서리스 동작을 위한 EEMF 옵저버도 구현되어 있습니다.

  • Clarke / Park 변환: 3상 정지 좌표계 → α-β → d-q 동기 좌표계
  • PI 전류 제어기: d축(자속), q축(토크) 독립 제어, Anti-windup 포함
  • PI 속도 제어기: 속도 기준 경사 제한(Ramp), Anti-windup 포함
  • SVPWM: min-max 오프셋 방식, NLC(비선형 보상) 전압 추가
  • Hall PLL 옵저버: 3비트 Hall 상태 → 전기각 매핑 → PI-PLL 속도 추정
  • EEMF 센서리스 옵저버: 확장 동기 좌표계 역기전력 기반 위치 오차 추정
  • 개루프 제어: 전류 개루프(OLC), 전압 개루프(VOLC), 전압 기준 생성 모드
